Nesse módulo, o objetivo é entender o funcionamento da linguagem **JavaScript**.

Através de exemplos práticos, são abordados aspectos da linguagem que são empregados no dia a dia do desenvolvedor web. Desde *variáveis* até o **DOM** - **Document Object Model** -, as atividades elaboradas para o módulo levam à compreensão desses conceitos e permitem o desenvolvimento de páginas, utilizando essa tecnologia.
